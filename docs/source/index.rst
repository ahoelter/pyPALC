Welcome to pyPALC's documentation!
==================================

This is a **Loudspeaker Array Optimization Software** to find the tilt angles of the *Loudspeaker Cabinets*.
The application is provided by the `Audio Communication Group <https://www.ak.tu-berlin.de>`_. The graphical user interface is based on `bokeh <https://www.bokeh.org>`_.
